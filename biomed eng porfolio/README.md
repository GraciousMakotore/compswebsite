# Submission Instructions — Biomedical Engineering Portfolio

Thank you for preparing this portfolio. Use this README as the single place to gather and submit your work.

## What to include
- A copy of the site (all HTML, CSS, images) in a ZIP file.
- A short PDF or plain-text summary for each project (1 page each) that includes: goal, methods, outcome, and your role.
- Any source code or schematics (zip or link to a repo) necessary to reproduce or evaluate the project.

## Preferred file names
- portfolio-site.zip — full site export (HTML, CSS, images).
- project-name_summary.pdf — short one-page summary for each project.
- project-name_sources.zip — optional supporting files or code.

## How to submit
Email your ZIP and supporting files to: graciousmakotore@gmail.com

Suggested email subject line: "Portfolio submission — [Your Name]"

In the body of the email please include:
- Your full name
- One-line summary of your background (3–4 words)
- A list of attached files and short notes for each (optional)

## Alternatives
If you prefer to use GitHub/GitLab, provide a link to the repository and add an archive (release) or a ZIP download.

## Checklist before submitting
- [ ] All pages render locally (open the `index.html` file in a browser).
- [ ] Images are included in the `images/` folder and load correctly.
- [ ] Contact information and links work.
- [ ] README and project summaries are attached.

If you want, I can create a ZIP of the site for you or prepare a short submission email template — tell me and I'll add it here or generate it for you.
